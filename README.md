# datathon-2026-career-success-regression
End-to-end ML pipeline predicting a censored career-success score from tabular + Turkish mentor-feedback text. GBDT/TabPFN/fine-tuned-BERT ensemble with year distribution-shift reweighting, a two-part censored model, and honest paired-CV validation. Private-LB MSE 83.75 (R²≈0.69).
